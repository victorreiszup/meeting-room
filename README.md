
# Meeting Room

API Rest desenvolvida, para agendamento de sala.

# Stack de tecnologia
- Spring Boot
    - Spring Web
    - Spring Data JPA
    - Spring Validation
    - Spring Boot Test
- Swagger
- Flyway
- MapStruct
- PostgreSQL
- Mokito
- Testcontainers
- AssertJ
- Docker

# Requesitos

- Java 8
- Docker
- Maven
- PostgreSQL




## Deploy

Para fazer o deploy desse projeto rode

```bash
  mvn clean install
  docker-compose -f docker-compose.yml up
```


## Teste

Para executar os testes desse projeto rode

```bash
  mvn test
  
```
